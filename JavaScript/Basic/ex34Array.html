배열의 주요 함수(메서드)

push(): 배열에 새로운 요소를 마지막에 추가한다.

unshift(): 배열에 맨 앞에 새로운 요소를 추가한다

concat(): 배열의 요소를 모두 합쳐 배열을 만들어 리턴한다.

join(): 배열 안의 모든 요소를 문자열로 만들어 반환한다.

pop(): 배열의 마지막 요소를 제거하고 반환한다.

shift(): 배열의 첫번째 요소를 제거하고 반환한다.

sort() : 오름차순으로 정렬하여 반환한다.

splice(start인덱스, [삭제할count수]): 시작인덱스에서 count수만큼 삭제한다

splice(startIndex, delCount, addElement): 삭제한 뒤 새로운 요소를 삽입한다

slice(start, end) : start인덱스에서부터 end-1까지의 요소를 조각내어 반환함

forEach(handler함수): 배열에 저장된 요소만큼 반복문을 돌면서 저장된 요소를 핸들러 함수의 매개변수에 전달함
